# 🍲 Food Project 1

A multi-page web project built to demonstrate the core fundamentals of HTML structure and page navigation.

## 📝 Project Overview
This project consists of a home page and a detailed recipe page. It was created using **Pure HTML** to practice document flow and linking without the use of external frameworks or CSS.

## 🚀 Features
* **Multi-Page Navigation:** Seamlessly linked `index.html` to `Recipe.html` using anchor tags.
* **Structured Content:** Used semantic tags like headings (`<h1>`, `<h2>`), unordered lists (`<ul>`), and ordered lists (`<ol>`) for clear readability.
* **Media Integration:** Embedded images with proper attributes and dimensions.

## 🛠️ Technologies Used
* **HTML5 Only:** Focused on mastering tags, attributes, and file paths.

## 📂 How to Navigate
1. Open `index.html` to see the recipe preview.
2. Click **"View Full Recipe"** to navigate to the detailed instructions in `Recipe.html`.
3. Use the **"Back to Home"** link to return to the main page.
